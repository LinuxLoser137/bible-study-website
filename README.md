# Bible Study Website

A beginner-friendly Bible study website built with HTML and CSS. This project helps organize KJV Bible verses, study notes, themes, and scripture topics.

## Live Website

soft-tapioca-d2e0bb.netlify.app

## About This Project

This project was created as a personal Bible study website using only KJV scripture. It includes multiple pages for organizing verses, notes, themes, and study content.

The goal of this project was to practice building a clean multi-page website while keeping the content organized, readable, and properly focused on Bible study.

## Features

- Home page introducing the website
- Verses page for KJV scripture references
- Notes page for Bible study notes
- Themes page for organizing verses by topic
- Navigation between pages
- Clean and simple layout
- KJV-focused content

## Technologies Used

- HTML
- CSS
- Git
- GitHub
- Netlify

## What I Learned

While building this project, I practiced creating a multi-page website, linking pages together, organizing written content, styling pages with CSS, and preparing a project for deployment. I also learned how to keep a project focused around a specific purpose and audience.
